PhtonView 是一款专业的 Android 相机遥控应用。通过 USB OTG 或实验性 WiFi 连接，实时取景、调整参数、触发快门，让拍摄更自由。

专业功能，简洁体验

为摄影师打造的随身遥控工具

实时取景

在手机上查看相机实时画面，支持峰值对焦与直方图辅助。

参数遥控

ISO、光圈、快门、曝光补偿、测光模式一手掌控。

多种拍摄模式

单拍、连拍、定时、B门、间隔拍摄、包围曝光应有尽有。

双连接方式

USB OTG 稳定低延迟，WiFi 连接摆脱线缆束缚。

界面模式

简洁模式快速上手，专业模式释放全部功能。
